# 分享功能备份说明

## 概述
本文档记录了历史记录页面和收藏页面的分享功能实现，以便后续恢复。

## 临时禁用原因
- 分享功能需要后端API支持才能让其他用户正常访问分享内容
- 目前缺少专用的分享内容获取API（如 `/api/shared/{id}`）
- 别人点击分享链接时会出现"分享的内容暂时无法访问"错误

## 需要的后端API
恢复分享功能需要以下后端接口：

```
GET /api/shared/{id}
- 用途：根据分享ID获取分享的内容详情
- 参数：id（分享记录的唯一标识）
- 返回：分享的完整内容信息

POST /api/share
- 用途：创建分享记录，返回分享ID
- 参数：内容数据
- 返回：分享ID和链接
```

## 已实现的功能特点

### 历史记录页面分享
- 每条记录都有分享按钮
- 详情弹窗中有分享按钮（替换原关闭按钮位置）
- 支持分享给微信好友和朋友圈
- 分享链接格式：`/pages/shared/shared?id=${item.id}&type=friend`

### 收藏页面分享  
- 每条收藏都有分享按钮
- 详情弹窗中有分享按钮
- 分享逻辑与历史记录页面一致
- 支持富文本内容分享

### 分享页面(shared)
- 完整的分享内容展示页面
- 富文本渲染支持
- 多种API获取策略
- 优雅的错误处理
